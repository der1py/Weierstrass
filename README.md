# Weierstrass

game made for ap calc :P
enemies are mathematical expressions and attacks can transform them

includes informative cutscenes about Karl Weierstrass

## Features

- WASD to move, mouse to shoot
- Fight enemies represented by expressions like `21`, `x`, or `x^2 + 4x + 4`
- math-based attacks including addition, subtraction, root checking, and derivatives
- 3 levels containing multiple waves
